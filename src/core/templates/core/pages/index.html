Advising — upcoming functionality

This is the planned booking flow for advisors and students. We’ll implement it incrementally.

Advisor dashboard

  • Calendar view shows the current month, with today highlighted.
  • Navigation to previous and next months.
  • Clickable dates take the teacher to a Day Detail page.

Day Detail (teacher view)

  • Timeline lists 15-minute slots from 06:00 to 23:00.
  • Advisors select a 15-minute slot to mark availability. Appointments are actually 30 minutes; selecting 08:00 marks both 08:00–08:15 and 08:15–08:30 as available in the same row.
  • Each availability requires a type: {% c badge %}Online{% endc %} {% c badge %}In person{% endc %} {% c badge %}Both{% endc %}
  • Selecting a slot toggles availability; selecting again unsets it (preventing accidental double-booking).

Student booking

  • Students browse advisors and see real-time availability.
  • Students can filter by date, time, and type (Online / In person / Both).
  • Booking a slot reserves a 30-minute appointment and blocks overlapping 15-minute cells.

Rules & constraints

  • Availability is stored in 15-minute granularity; bookings consume two adjacent slots.
  • Overlaps are prevented; adjacent same-teacher bookings must not clash.
  • Availability type is required and may be validated against location/video settings.
